Type Race

  • Originality: A typing game where you race against the clock to get to the finish line before time is up.
  • Technical Merit: My game uses a timer to add a challenge for how much time the player has to finish the race. I use a console prompt to have the player type in words. I added the car in as a sprite so it moves everytimem the player types a word correctly. The car accelerates as you get to the end. I added music and a reset key to make the game more appealing.
  • Prototype Postmortem: The hardest part about this game was figuring out how to get user input. Also how to use a key press to get the console prompt. I learned how to assign functions to key presses.

  • High Concept: Finish the race before time is up by typing words fast!
  • Theme: State which of the Storymatic themes you used.
  • Mandated Variety: Direct keyboard inputs, deterministic, competitor, racing game.
  • Prototype Goal: I wanted to make a typing game since I really enjoy playing those to better my typing skills and speed.
  • Player Experience Goals: I want players to have fun testing their typing skills.
  • Gameplay: The player must press the enter key to bring up the console prompt. The player must then type the word they are told to. If they type it in wrong then nothing happens. If they type it in correctly then the car moves closer to the finish line. The win if they reach the finish line before time is up. They lose if the time is over before they can finish.
  • Strategies: The player will have to type fast in order to finish the race in time. The words don't change so they can learn what words are going to be presented and learn how to type those quickly.
  • Story/Setting/Premise: The player must help the car finish the race by typing words quickly.
  • Target Audience: People who enjoy testing their typing skills.
  • Play Time: 0 - 20 seconds.
