Rhyme Time!

  • Originality: Type as many rhyming words as you can before time runs out!
  • Technical Merit: Used a timer with countdown, used string array for rhyming word, used prompt to get player input, used button with click action
  • Prototype Postmortem: I learned how to use a countdown timer with this game. I also learned how to make string arrays in javascript. I tried to find an easy way to implement player typing text input and the prompt was the easiest way I could find.
  • Prototype Assets:
    • Timer: source code
    • Background: background
    • Pointer Code: phaser code
  • Prototype Closest Other Game: I could not find a similar game.
  • High Concept: Type as many rhyming words as you can before time is up!
  • Theme: Poet Theme
  • Mandated Variety: The game is driven by the player typing in as many words as they can into the prompt using their keyboard. There is no randomness in the game. I think the player type attracted to this game would be Artists who enjoy being creative with rhymes.
  • Prototype Goal: I wanted to focus less on technical merrits and more on creating a unique game. A lot of the stuff I used here was new so it took me a while to get it working. Most of my past games were driven by what I knew how to code and this is my first game driven by an idea that I wanted to execute even if I didn't know how to code it at first.
  • Player Experience Goals: I want the player to have fun while they avoid bombs and collect vials and try to get a high score.
  • Gameplay: The player is given a word and they must click the button and type in as many rhyming words as they can before the timer reaches 0.
  • Strategies: The player can play the game over and over again as they come up with more rhyming words. The challenge becomes how fast they type to reach a higher score if they know enough words.
  • Story/Setting/Premise: The stage lights and mic are on. The crowd is watching. Get up on stage and perform as many rhyming words as you can before your performance time slot ends.
  • Target Audience: My audience is people who enjoy poems and making rhymes. People who like typing games would also be attracted to this game.
  • Play Time: 30 second timer.
